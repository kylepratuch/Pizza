# Pizza

##### Price out a pizza order according to chosen size, toppings, and quantity. 9/18/15

#### By _Kyle Pratuch_

## Description

_Pizza allows a user to generate a pizza order and price it out with the click of a button!_

## Setup

 * Clone this repository: run command ```git clone https://github.com/kylepratuch/Pizza.git ``` then change directory to top level of project folder.
 * Open pizza.html in your browser.
 * To run the included tests, open spec-runner.html in your browser.

## Technologies Used

_HTML, CSS, JavaScript, Mocha, Chai, jQuery_
